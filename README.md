# Config Validator

A simple and flexible configuration validation tool built using Python.  
This project helps validate configuration files against predefined schemas to ensure correctness, consistency, and reliability.

---

##  Problem Statement
Configuration files (JSON, YAML, ENV) are widely used in applications.  
Invalid or missing configurations can lead to runtime errors and system failures.

This project validates configuration files by checking:
- Required fields
- Data types
- Structure correctness
- Schema compliance

---

## Features
- Supports JSON, YAML, and ENV configuration files
- Schema-based validation
- Easy to extend
- Beginner-friendly Python project
